## §3. 데이터 구조 (JSON 스키마)

> 개발팀은 `src/data/tripData.ts` 단일 파일로 모든 콘텐츠를 관리한다. 이 섹션의 필드명·구조를 그대로 따를 것.

```typescript
// src/data/tripData.ts

interface TripData {
  meta: TripMeta
  flights: Flight[]
  accommodation: Accommodation
  days: Day[]
  transport: TransportLeg[]
  tips: Tips
  japaneseRequests: JapaneseRequest[]
}

interface TripMeta {
  appTitle: string          // 앱 표시 제목
  travelers: string[]       // ["CHUN SUNGBUHM", "CHUN JIYONG"]
  startDate: string         // "2026-08-06"
  endDate: string           // "2026-08-09"
  region: string            // "쓰루오카·사카타"
}

interface Flight {
  id: string                // "outbound" | "return"
  flightNumber: string      // "KE2197"
  departure: { airport: string; code: string; datetime: string }
  arrival: { airport: string; code: string; datetime: string }
  seats: string[]           // ["33A", "33F"] / ["33B", "33E"]
}

interface Accommodation {
  name: string              // "쇼나이 호텔 스이덴 테라스"
  nameJa: string            // "庄内ホテル スイデンテラス"
  address: string
  phone: string             // "+81 235-25-7424"
  bookingRef: string        // "73429589556696"
  bookingService: string    // "Hotels.com"
  checkIn: string           // "2026-08-06"
  checkOut: string          // "2026-08-09"
  nights: number            // 3
  mapQuery: string
}

interface Day {
  day: number               // 1~4
  date: string              // "2026-08-06"
  dayOfWeek: string         // "목"
  theme: string             // "이동 + 쓰루오카 문화 클러스터"
  events: Event[]
}

interface Event {
  id: string
  time: string              // "09:40"
  title: string             // 한글 표기
  titleJa: string           // 일본어 표기
  type: "transport" | "sightseeing" | "meal" | "activity" | "accommodation"
  description: string
  mapQuery?: string
  cost?: string             // "¥1,250" 또는 "무료"
  duration?: string         // "약 54분"
  alert?: string            // ⚠ 주의 사항
  guide?: EventGuide        // 바텀시트 상세 안내 (선택)
}

interface EventGuide {
  summary: string           // 한 줄 요약
  description: string       // 본문 (\n 줄바꿈 처리)
  tips?: string             // 팁 (있을 경우에만)
}

interface TransportLeg {
  id: string
  from: string
  to: string
  mode: "bus" | "train" | "taxi" | "flight" | "walk"
  modeLabel: string         // "이나호 특급"
  duration: string          // "1시간 20분"
  cost: string              // "¥2,640"
  note?: string
}

interface Tips {
  weather: string           // 오프라인 폴백용 정적 날씨 정보
  payment: string
  misc: string[]
}

interface JapaneseRequest {
  id: string
  label: string             // 화면 표시 레이블
  text: string              // 복사 대상 일본어 전문
}
```

## §7. 인터랙션 정의

### 7-1. 클립보드 복사

**대상**: 예약번호(숙소·항공), 일본어 안내 문구

**동작**:
1. 복사 대상 텍스트 또는 복사 아이콘 탭
2. `navigator.clipboard.writeText()` 호출
3. 토스트 표시: "복사됨 ✓" (1.5초 후 자동 소멸)
4. 복사 아이콘 순간 체크 아이콘으로 전환 후 복귀

**탭 영역**: 최소 48×48dp

### 7-2. 일정 DAY 전환

**형태**: 수평 세그먼트 컨트롤 (DAY 1 | DAY 2 | DAY 3 | DAY 4)

**동작**:
- 탭하면 해당 DAY 타임라인 전환
- 활성 DAY: primary-container 배경 + 흰색 텍스트
- 비활성 DAY: surface-container 배경 + on-surface-variant 텍스트
- 오늘 날짜 기준 자동 선택 (여행 전/후에는 DAY 1 기본)

### 7-3. 이벤트 안내 바텀시트

**트리거**: guide 필드가 있는 이벤트 카드 탭

**구성**:

| 영역 | 내용 |
|------|------|
| 핸들 | 드래그 핸들 바 (중앙 상단) |
| 헤더 | 이벤트 title + guide.summary |
| 본문 | guide.description (줄바꿈 처리) |
| 하단 | guide.tips (있을 경우만, 팁 박스) |

**닫기**: 아래 스와이프 / 외부 탭 / X 버튼
**최대 높이**: 화면의 85%

### 7-4. ⚠ 주의 항목 강조

- 카드 좌측 보더: error 컬러 (#ba1a1a 계열)
- 경고 텍스트 카드 하단 별도 박스
- alert 있는 이벤트에만 적용

### 7-5. 오프라인 상태 표시

- `navigator.onLine` 감지
- 오프라인 시: 상단 얇은 배너 "오프라인 모드 · 모든 정보 이용 가능"
- 온라인 복귀 시: 배너 페이드 아웃
